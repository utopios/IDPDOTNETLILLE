> Ce server sert à l'utilisation de l'Application developpé avec React : [REACT-IMC](https://gitlab.com/mohamed.aijjou1780/calcul-imc-react). Cette infrastructure web a été developpé avec [Express](https://expressjs.com/fr/).

## Description

Ce server n'est pas relié avec une base de donnée. Il est composé pour la partie persistence de fichiers json qui sont modifiés par le fichier intitulé index.js en lecture ou en ecriture.

## Getting Started

### Dependencies && Build With

- NodeJs
- Express
- Cors
- Body-parser

### Installing

##### 1 - Front part :

- Step 1 : ``` git clone https://gitlab.com/mohamed.aijjou1780/calcul-imc-react.git ```
- Step 2 : ``` npm install ```
- step 3 : ``` npm start ```

##### 2 - Back part (partie actuelle):

- Step 1 : ``` git clone https://gitlab.com/mohamed.aijjou1780/server-nodejs-express-calcul-imc.git ```
- Step 2 : ``` node index.js ```
